# PRIS-2026-DaniyarOrazov-ArtGalleryAI

## Title
**ArtGalleryAI** — виртуальная художественная галерея с элементами
интеллектуальных рекомендаций.

---

## Description
Проект решает проблему удобного и персонализированного доступа
к произведениям искусства.  
Пользователю сложно ориентироваться в больших коллекциях художественных работ,
поэтому система предоставляет поиск, структурирование и рекомендации
на основе интересов пользователя.

---

## Tech Stack
В проекте планируется использовать следующие технологии и библиотеки:

- Python  
- Streamlit (пользовательский интерфейс)  
- Pandas (обработка и анализ данных)  
- Git и GitHub (контроль версий)  
- Visual Studio Code (среда разработки)  

---
